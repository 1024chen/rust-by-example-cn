在前面的例子中，我们使用组合算子显式地处理错误。 另一种处理这种情形分解的方法是使用 `match` 语句和**提前返回**（*early returns*）的组合形式。

现在我们已经学会了使用组合算子和提前返回来显式地处理错误。虽然我们通常希望避免 `panic`，但显式处理我们所有的错误将会麻烦。

在下一节，我们将介绍 `try!`，用在这样的场景，我们只需要简单地 `unwrap` 而避免可能的 `panic`。
