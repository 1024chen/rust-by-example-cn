结构体（structure，缩写成 struct）有 3 种类型，使用 `struct` 关键字来创建：

* 元组结构体，总的来说是根据元组来命名。
* C 语言传统的结构体[c_struct]。
* 单元结构体，不带字段，在泛型中很有用。
